# Booking

## Description
A simple book tracking app where it stores the book name and total amount of pages of said book.

## Features
1. Rename
- The user can rename any book from the database (located on books.json).
2. Update
- They can change the total amount of pages on any book from the database.
3. Remove
- They can also delete a book's entry from the database.
4. Remove All
- Last but not the least, the user can delete every book data from the database
